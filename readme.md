# ARB Exam

## Installation

1. Create arb_calls database
2. In root directory, run the following command

```bash
run composer
```
```bash
npm install
```
3. Set DB Credentials in env
2. In root directory, run the following command

```bash
php artisan migrate
```
```bash
php artisan db:seed
```
```bash
php artisan passport:install
```
